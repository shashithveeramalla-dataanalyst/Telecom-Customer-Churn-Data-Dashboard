# Telecom Customer Churn Analysis Dashboard | Power BI


### 📌 Overview
This project focuses on analyzing telecom customer churn behavior using an interactive Power BI dashboard. The objective is to identify patterns and key factors contributing to customer churn and provide insights that can help improve customer retention strategies. By leveraging customer demographics, service usage, and account information, the dashboard enables data-driven decision-making for reducing churn.


### ⭐ 📌 Key Features & Highlights
# 📊 Business KPIs
- Tracked 7K+ customers with real-world telecom data
- Calculated Churn Rate (26.5%) using DAX
- Built KPI cards for:
- Total Customers
- Churned Customers
- Retained Customers

### 📉 Churn Analysis
- Identified highest churn in month-to-month contracts
- Highlighted early churn risk (0–6 months tenure ~53%)
- Visualized churn distribution using interactive charts

### 🔍 Customer Segmentation
- Segmented customers by:
  Contract type
  Internet service
  Tenure bands
- Enabled deeper analysis of customer behavior patterns

### 💰 Revenue Insights
- Analyzed Monthly Charges vs Customer Retention
- Identified pricing patterns influencing churn

### 🎨 Dashboard Design
- Built a modern dark-themed dashboard UI
- Clean layout with intuitive navigation
- Interactive filters for dynamic exploration
